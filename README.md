Welcome to the Movie Matcher!

I developed a movie recommender using Python, Scikit-Learn, Pandas and Streamlit. The system is using a dataset providing general information about 5000+ movies and based on the movie title the user provides, it finds top 6 best fits. The general features taken into consideration when computing the similarty scores are the movie's director, actors, main genre and side genres.
The user is able to provide the maximum runtime in minutes the recommendations should have as well as decinding through percents if for them is more important the similarity with the provided movie or the ranting. Don't worry about misspelling, RapidFuzz fixes this problem.

A basic graphical interface using Streamlit is provided. At the link below you can find the live demo.
Enjoy watching!
